# 互动点子库

## 运行说明

### 在微信开发者工具中导入运行

1. **下载并安装微信开发者工具**
   - 访问 [微信开发者工具官网](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载
   - 安装并打开微信开发者工具

2. **导入项目**
   - 点击"导入项目"或"+"号创建新项目
   - 项目目录选择：`/Users/cuijian123/mygame`（当前项目文件夹）
   - AppID 填写：
     - 如有小程序 AppID，填写真实的 AppID
     - 如无 AppID，选择"测试号"或"不使用云服务"（不影响本地开发调试）
   - 项目名称：互动点子库（可自定义）
   - 点击"确定"即可运行

3. **预览调试**
   - 在开发者工具中可以直接查看效果
   - 点击工具栏的"预览"按钮，用微信扫码可以在手机上查看实际效果

## 功能说明

### 核心功能
1. **场景选择**：支持"过年"和"结婚"两种喜庆场景
2. **随机生成**：从选定场景中随机选择一款游戏
3. **指定生成**：点击游戏名称，查看指定游戏的详细规则
4. **换一个**：保留当前场景，重新随机生成游戏
5. **重置**：清空所有选择，回到初始状态

### 游戏数据
- **过年场景**：包含4款游戏（抢红包、新年祝福接龙、年货大采购、爆竹传花）
- **结婚场景**：包含4款游戏（爱情密码、红包雨、花式敬酒、婚礼寻宝）

每个游戏包含：
- 游戏名称
- 游戏描述
- 适合人数
- 所需道具列表
- 详细规则列表

## 技术说明

- **开发框架**：微信小程序原生开发
- **兼容版本**：支持微信小程序基础库 2.0 及以上版本
- **页面数量**：单页面应用，所有功能在 index 页面完成
- **代码特点**：包含详细中文注释，符合微信小程序开发规范

## 广告说明（激励视频）

- **触发时机**：每次「随机生成」「换一个」或点击「指定游戏」前，需观看一段激励视频广告，看完后才执行对应操作；未看完则提示「请观看完整广告后再试」。
- **配置广告位**：在 `pages/index/index.js` 顶部将 `REWARDED_VIDEO_AD_UNIT_ID` 替换为你在微信公众平台申请的激励式视频广告位 id。
  - 路径：登录 [微信公众平台](https://mp.weixin.qq.com) → 流量主 → 广告管理 → 激励式视频广告 → 创建广告位 → 获取 `adUnitId`。
- **测试**：未配置或广告位未审核通过时，会提示「广告加载失败，请稍后重试」；真机预览时需使用已绑定的正式/体验版小程序，才能拉取到广告。

## 注意事项

1. 首次运行需要确保微信开发者工具版本为最新
2. 建议使用真机预览功能查看实际显示效果
3. 代码已适配不同屏幕尺寸，支持主流手机屏幕
4. 激励视频广告需小程序开通流量主并成功创建激励视频广告位后才可正常展示
